uplus

Purpose

Unitary plus for a polyhedron.

Syntax

Q = +P;

Description

Creates a new copy Q of the polyhedron P by copying all fields.

Input Arguments

P

Polyhedron object.

Class: Polyhedron

Example(s)

Example 1

Create H-polyhedron
 P = ExamplePoly.randHrep; 
 Copy the polyhedron
 Q = +P; 


 
         P and Q are the same.
 P==Q 


ans =

     1
