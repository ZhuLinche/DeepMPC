isEmptySet

Purpose

Test if a polyhedron has a non-empty interior.

Syntax

tf = P.isEmptySet
tf = isEmptySet(P)

Description

Return true if the polyhedron is empty or false otherwise. A polyhedron is considered as empty if the diameter of the inscribed ball is less than region_tol or if there exist no feasible point inside the set.

Input Arguments

P

Polyhedron in any format

Class: Polyhedron

Output Arguments

tf

True if the polyhedron is empty and false otherwise.

Class: logical

Allowed values:

  • true
  • false

Example(s)

Example 1

Non-empty polyhedron

ans =

     0
