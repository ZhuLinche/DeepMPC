isPointed

Purpose

Test if a polyhedron is pointed or not

Syntax

ts = P.isPointed
ts = isPointed(P)

Description

A polyhedron P in H-representation is pointed if and only if its lineality space null([A; A_e]) is empty. If the polyhedron is in V-representation, automatic conversion to H-representation will be performed.

Input Arguments

P

Polyhedron in any format

Class: Polyhedron

Output Arguments

ts

True if the polyhedron is pointed and false otherwise.

Class: logical

Allowed values:

  • true
  • false

Example(s)

Example 1

The unpointed polyhedron does not contain any vertex. For instance the following low-dimensional polyhedron is un-pointed: 
 A = [-1.2 1.63; 0.7 0.5]; b = [1; 0.7]; Ae = [0.4 -1.4]; be = 0.8;


 P = Polyhedron('A',A,'b',b,'Ae',Ae,'be',be); 


The polyhedron P is pointed
P.isPointed


ans =

     1
