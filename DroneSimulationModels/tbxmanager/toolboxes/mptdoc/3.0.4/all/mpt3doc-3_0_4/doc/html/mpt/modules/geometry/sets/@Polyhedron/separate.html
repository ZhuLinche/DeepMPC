separate

Purpose

Separate a point/polyhedron from another polyhedron.

Syntax

h = P.separate(x)
h = separate(P, x)
h = P.separate(S)
h = separate(P, S)

Description

Computes a separating hyperplane between and / :

or

Input Arguments

P

Polyhedron in any format

Class: Polyhedron
S

Polyhedron in any format

Class: Polyhedron
x

Column vector of length P.Dim.

Class: Polyhedron

Output Arguments

h

Separating hyperplane , or [] if none exists.

Class: double

Example(s)

Example 1

Create a Polyhedron:
P = ExamplePoly.poly3d_sin;
Choose a point:
x = [2;2];


Separate and plot
h = P.separate(x);
	 P.plot; hold on; pplot(x,'ro');
	 Polyhedron('He',h).plot('linewidth', 2);
