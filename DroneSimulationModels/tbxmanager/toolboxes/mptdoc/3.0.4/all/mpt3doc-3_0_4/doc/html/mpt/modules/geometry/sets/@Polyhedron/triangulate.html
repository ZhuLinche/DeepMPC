triangulate

Purpose

Triangulation of a polyhedron.

Syntax

T = P.triangulate;
T = triangulate(P)

Description

The polyhedron is split into an array of triangular polyhedra that built a partition . The triangulate function works over full-dimensional polyhedra, that are bounded and not empty.

Input Arguments

P

Polyhedron object.

Class: Polyhedron

Example(s)

Example 1

Create V-polyhedron
 P = 10*Polyhedron(randn(5,2)); 
 Generate the triangular partition
 T = P.triangulate; 


 Plot the triangulation. 
 T.plot
