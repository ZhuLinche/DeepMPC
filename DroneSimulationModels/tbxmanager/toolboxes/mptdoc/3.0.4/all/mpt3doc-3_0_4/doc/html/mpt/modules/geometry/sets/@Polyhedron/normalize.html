normalize

Purpose

Normalizes polyhedron in H-representation.

Syntax

P.normalize
normalize(P)

Description

Normalize polyhedron by scaling each inequality and equality such that each row has unitary norm, i.e. and . The normalization routine is very useful to prevent from numerical problems involved in badly scaled polyhedra.

Input Arguments

P

Polyhedron object.

Class: Polyhedron

Example(s)

Example 1

Badly scaled polyhedron 
 P = Polyhedron([1000, 0.4, -351; -0.1, -1e4, 89.1],[780; 0.067]); 
 The values are stored inside the polyhedron as they are provided.

                      -0.1                    -10000                      89.1                     0.067
