minHRep

Purpose

Compute an irredundant H-representation of a polyhedron.

Syntax

P.minHRep()
[P, sol] = P.minHRep()

Description

Computes an irredundant H-representation of the polyhedron:

Notes:
  • If an H-representation is already known, then this function does redundancy elimination.
  • Calling P.minHRep() will store the irredundant H-representation in .
  • For empty polyhedron the result remains the same (no redundancy elimination).

Input Arguments

P

Polyhedron in any format

Class: Polyhedron

Output Arguments

sol

Irredundant H-representation of

Class: struct

sol.H

Matrix of inequalities

Class: double matrix
sol.He

Matrix of equalities

Class: double matrix

Example(s)

Example 1

Create redundant description of a polyhedron:
P = Polyhedron('H', [randn(20,3) ones(20,1)], 'He', [randn(1,3) 0])
Polyhedron in R^3 with representations:
    H-rep (redundant)   : Inequalities  20 | Equalities   1
    V-rep               : Unknown (call computeVRep() to compute)
Functions : none
Remove redundancies:
P.minHRep()
Polyhedron in R^3 with representations:
    H-rep (irredundant) : Inequalities   6 | Equalities   1
    V-rep               : Unknown (call computeVRep() to compute)
Functions : none
Plot the result
plot(P);


The polyhedron P has been permanently changed:
disp(P)
