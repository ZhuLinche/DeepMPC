mpt_call_mpqp

Purpose

A gateway function to MPQP solver (without errorchecks)

Syntax

R = mpt_call_mpqp(S)

Description

The function call to MPQP solver from Opt class. Using option settings for MPQP solver taken from MPT2.6.
